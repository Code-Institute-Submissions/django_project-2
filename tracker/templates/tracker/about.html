About Unicorn

Unicorn is a web app aimed at fixing bugs and creating new features for our service. Users are welcome and can register to start using our services. Unicorn allows users to log their tickets which will appear on the list of tickets on the homepage. We will fix these bugs for free and we will use the popularity of bugs to prioritize fixes. To help us decide you the user can click the upvote symbol in post descriptions. We also provide a features service which is similar to our bug fixing service, you the user can suggest a new feature for us to develop but to enable us to develop a feature we need to charge a fee to upvote a feature request. If a feature request gets a certain number of upvotes we will develop the feature fo our users.
